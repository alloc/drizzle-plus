# Universal SQL functions

These functions are available in all dialects, since they're part of the SQL standard.

- **Syntax:**
  - `caseWhen`
  - `nest`
  - `toSQL`
- **SQL functions:**
  - `abs`
  - `ceil`
  - `coalesce`
  - `concatWithSeparator`
  - `currentDate`
  - `currentTime`
  - `currentTimestamp`
  - `floor`
  - `length`
  - `lower`
  - `mod`
  - `nullif`
  - `power`
  - `round`
  - `sqrt`
  - `substring`
  - `trim`
  - `upper`

Import them from the `drizzle-plus` module:

```ts
import { caseWhen } from 'drizzle-plus'
```
